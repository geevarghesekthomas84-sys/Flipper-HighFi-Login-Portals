# Flipper-HighFi-Login-Portals


<img width="78" height="20" alt="image" src="https://github.com/user-attachments/assets/e7eae45b-2dda-4e76-8f83-b1e3c99dd852" />

  A frontend research project exploring self-contained authentication-style interfaces designed for constrained captive-portal    
  environments. This repository contains HTML/CSS/JavaScript mockups inspired by widely recognized sign-in flows, built to study  
  responsive layout systems, inline assets, interaction states, and offline rendering behavior.

  ## Overview

  This project focuses on recreating polished, production-style portal interfaces using only local resources. Each template is    
  packaged as a standalone page so it can render reliably in limited environments where external fonts, scripts, and asset CDNs   
  may not be available.

  The repository is useful for:
  - Frontend developers studying complex login UI patterns
  - UI/UX researchers comparing interaction flows across brands
  - Security researchers analyzing captive-portal presentation and browser behavior                                               
  - Designers interested in responsive split layouts, floating labels, and embedded SVG branding                                  
                                                                                                                                  
  ## Included Templates                                                                                                           
                                                                                                                                  
  The project currently includes multiple standalone variants inspired by major web login experiences:                            
                                                                                                                                  
  - Google                                                                                                                        
  - Apple                                                                                                                         
  - Facebook                                                                                                                      
  - Amazon                                                                                                                        
                                                                                                                                  
  Each variant lives in its own folder and is structured as a self-contained `index.html`.                                        
                                                                                                                                  
  ## Features
                                                                                                                                  
  - Standalone single-file templates                                                                                              
  - Inline CSS and JavaScript                                                                                                     
  - Responsive desktop/mobile layouts                                                                                             
  - Multi-step sign-in style flows                                                                                                
  - Embedded vector branding and icons                                                                                            
  - Offline-friendly asset strategy                                                                                               
  - Variants for different frontend data-handling demonstrations

  ## Technical Notes                                                                                                              
                                                                                                                                  
  These templates are intentionally lightweight and self-contained:                                                               
                                                                                                                                  
  - HTML, CSS, and JavaScript are embedded directly in each page                                                                  
  - External dependencies are avoided for portability                                                                             
  - Layouts are tuned for portal-like browser contexts                                                                            
  - UI interactions are implemented with vanilla JavaScript
